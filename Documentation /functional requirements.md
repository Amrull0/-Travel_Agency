### Модуль: Управление турами (TOU)

| Функция                | ID      | Описание                                                                          | Приоритет    | Статус       | Доступ | Комментарий                                      |
| :--------------------- | :------ | :-------------------------------------------------------------------------------- | :----------- | :----------- | :----- | :----------------------------------------------- |
| **Просмотр туров**     | TOU-001 | Показ списка туров с основными данными: страна, город, длительность, цена.        | Обязательный | Реализовано  | Все    | Главный экран модуля.                            |
| **Добавить тур**       | TOU-002 | Создание нового тура с указанием направления, длительности, стоимости и описания. | Обязательный | Реализовано  | Администратор    | Только для сотрудников с правами редактирования. |
| **Редактировать тур**  | TOU-003 | Изменение данных существующего тура (цена, описание, даты).                       | Обязательный | В разработке | Администратор     | Позволяет исправлять ошибки.                     |
| **Удалить тур**        | TOU-004 | Удаление тура из базы с подтверждением.                                           | Желательно     | В разработке | Администратор    | Требуется подтверждение перед удалением.         |
| **Поиск/фильтр туров** | TOU-005 | Поиск туров по стране, цене и длительности.                                       | Обязательный | Реализовано  | Все    | Ускоряет подбор тура для клиента.                |
| **Детали тура**        | TOU-006 | Просмотр полной информации о туре, включая описание и стоимость.                  | Желательно     | Реализовано  | Все    | Используется при бронировании.                   |

### Модуль: Управление клиентами (CLI)

| Функция                   | ID      | Описание                                                                    | Приоритет    | Статус       | Доступ        | Комментарий                                                |
| :------------------------ | :------ | :-------------------------------------------------------------------------- | :----------- | :----------- | :------------ | :--------------------------------------------------------- |
| **Просмотр клиентов**     | CLI-001 | Отображение списка всех клиентов с основными данными: ФИО, телефон, email.  | Обязательный | Реализовано  | Все           | Главный экран модуля.                                      |
| **Добавить клиента**      | CLI-002 | Создание новой записи клиента с обязательными данными: ФИО, телефон, email. | Обязательный | Реализовано  | Администратор | Доступно только сотрудникам с правами редактирования.      |
| **Редактировать клиента** | CLI-003 | Изменение данных существующего клиента (ФИО, контакты).                     | Обязательный | В разработке | Администратор | Позволяет исправлять ошибки в данных.                      |
| **Удалить клиента**       | CLI-004 | Удаление клиента из базы данных с подтверждением.                           | Желательно     | В разработке | Администратор | Требуется подтверждение перед удалением.                   |
| **Поиск клиентов**        | CLI-005 | Поиск клиентов по ФИО или телефону.                                         | Обязательный | Реализовано  | Все           | Используется для быстрого поиска клиента при бронировании. |
| **Детали клиента**        | CLI-006 | Просмотр полной информации о клиенте, включая историю бронирований.         | Желательно     | Реализовано  | Администратор | Важная функция для анализа активности клиента.             |

### Модуль: Бронирование туров (RES)

| Функция                        | ID      | Описание                                                                      | Приоритет    | Статус       | Доступ        | Комментарий                                       |
| :----------------------------- | :------ | :---------------------------------------------------------------------------- | :----------- | :----------- | :------------ | :------------------------------------------------ |
| **Создать бронирование**       | RES-001 | Оформление нового бронирования для выбранного клиента и тура.                 | Обязательный | Реализовано  | Администратор | Фиксируется дата и статус бронирования.           |
| **Просмотр бронирований**      | RES-002 | Отображение списка всех бронирований с информацией о клиенте, туре и статусе. | Обязательный | Реализовано  | Все           | Главный экран модуля бронирования.                |
| **Редактировать бронирование** | RES-003 | Изменение данных существующего бронирования (тур, клиент, статус).            | Желательно     | В разработке | Администратор | Используется для корректировки ошибок.            |
| **Отменить бронирование**      | RES-004 | Отмена бронирования с автоматическим изменением статуса.                      | Желательно    | В разработке | Администратор | Требуется подтверждение перед отменой.            |
| **Поиск бронирования**         | RES-005 | Поиск бронирований по имени клиента, дате или туру.                           | Обязательный | Реализовано  | Все           | Ускоряет работу с большими списками бронирований. |
| **Детали бронирования**        | RES-006 | Просмотр полной информации о бронировании, включая клиента, тур и дату.       | Желательно     | Реализовано  | Администратор | Полезно для анализа и контроля бронирований.      |

### Модуль: Отчёты и статистика (REP)

| Функция                 | ID      | Описание                                                         | Приоритет    | Статус       | Доступ        | Комментарий                                |
| :---------------------- | :------ | :--------------------------------------------------------------- | :----------- | :----------- | :------------ | :----------------------------------------- |
| **Список бронирований** | REP-001 | Вывод всех бронирований с клиентами, турами, датами и статусами. | Обязательный | В разработке | Администратор | Для анализа текущих бронирований.          |
| **Фильтр по дате**      | REP-002 | Фильтрация бронирований по дате оформления.                      | Желательно     | В разработке | Администратор | Удобно для формирования отчетов за период. |
| **Статистика по турам** | REP-003 | Подсчет количества бронирований по каждому туру.                 | Желательно     | В разработке | Администратор | Позволяет оценивать популярность туров.    |
| **Поиск по клиенту**    | REP-004 | Вывод бронирований конкретного клиента.                          | Желательно    | В разработке | Администратор | Быстрый поиск истории клиента.             |
